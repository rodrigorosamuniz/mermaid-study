# Diagramas Mermaid para estudo

Esta pasta contém diagramas em formato `.mmd` para estudo no **Mermaid Live Editor**.

## Arquivos disponíveis

- `Banking.mmd` — diagrama de fluxo de um sistema bancário
- `Study_case.mmd` — estudo de caso com fluxo de ataque envolvendo SSRF, IMDS, IAM Role e S3

## Como usar

1. Abra o Mermaid Live Editor.
2. Abra um dos arquivos `.mmd` deste diretório em qualquer editor de texto.
3. Copie todo o conteúdo do arquivo.
4. No Mermaid Live Editor, cole o conteúdo na área **Code**.
5. O diagrama será renderizado automaticamente na área de **Preview**.
6. Faça alterações, se quiser, para testar cenários e entender melhor o fluxo.

## O que observar no Mermaid Live

- **Code**: área onde fica o código Mermaid
- **Preview**: visualização do diagrama gerado
- **Configuration**: opções de configuração visual
- **History**: histórico local de edições salvas no navegador

## Sugestão de estudo

- Primeiro visualize o diagrama sem alterar nada
- Depois identifique os componentes principais
- Em seguida, altere nomes, fluxos e conexões para ver o impacto visual
- Use isso para revisar arquitetura, fluxo de dados e possíveis riscos de segurança

## Observações

- O histórico do Mermaid Live Editor fica salvo apenas no armazenamento do navegador
- Você também pode exportar os diagramas em formatos como PNG, SVG e Markdown
- Caso o diagrama não apareça corretamente, revise a sintaxe e a indentação do código Mermaid
